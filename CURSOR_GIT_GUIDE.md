# 📝 How to Push to GitHub in Cursor

## Step-by-Step Process

### Step 1: Stage Your Changes
In the Source Control panel (Git tab):
- You'll see a list of modified files
- Click the **"+"** icon next to each file, OR
- Click **"Stage All Changes"** (or `Cmd+Shift+A`) to add everything at once

### Step 2: Commit
- Type a commit message in the box at the top (e.g., "Update app with latest fixes")
- Click **"Commit"** button
- This saves your changes locally

### Step 3: Push to GitHub
- After committing, you'll see a **"Push"** or **"Sync"** button appear
- Click it to send your code to GitHub
- Cursor will use your GitHub connection for authentication

---

## Quick Summary
1. **Stage** → Add files to commit
2. **Commit** → Save changes locally  
3. **Push** → Send to GitHub

---

## Alternative: Stage All & Commit in One Go
You can also:
- Click "Stage All Changes"
- Type commit message
- Click "Commit"
- Then click "Push"

That's it! 🚀
